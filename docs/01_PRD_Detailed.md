# PRD (상세)

## 프로젝트명
MediInsight

## 목표
공개 쇼핑몰 및 리뷰 데이터를 수집하여 근거 기반 보고서와 인스타그램 콘텐츠를 생성하고, Law MCP를 이용해 최종 광고 문구를 검증한다.

## 문제 정의
- 리뷰가 자사몰, 올리브영, 네이버, 쿠팡 등에 분산
- 채널별 고객 특성이 달라 단일 채널 분석은 편향 가능
- 콘텐츠 제작 시 법률 검토가 수작업

## 성공 기준
- 공개 데이터만 사용
- 모든 주장에 근거 리뷰 포함
- Law MCP 검수 통과
- 보고서 + 캐러셀 3세트 + 제품 홍보 이미지 생성
